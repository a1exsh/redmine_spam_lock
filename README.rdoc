= redmine_spam_lock plugin

This plugin adds a 'Lock' link next to 'Activate' for registered inactive users in the User Administration page on Redmine.

This way you don't need to activate a spammer user only to be able to lock it.

For Redmine versions prior to 2.x, check out the pre-2.x branch.
